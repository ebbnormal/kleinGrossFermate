== Klein Gross Fermate

A simple web application to construct models for generating text based scores. The models are built using a simple Naive Bayes classifier
The musical and compositional inspiration draws on the text-based compositions of Karlheinz Stockhausen, George Lewis, and Robert Ashley. 


Confer: https://en.wikipedia.org/wiki/Naive_Bayes_classifier.

See the app in action: https://kleingrossfermate.herokuapp.com

*  Ruby Version 2.1.1
